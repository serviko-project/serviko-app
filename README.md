# Serviko App

Serviko is a full-stack, AI-powered service marketplace mobile application built with Flutter. It enables users to discover, book and manage local services while allowing providers to accept jobs, manage availability and track earnings — all within a single unified application.

---

## 📱 Overview

Serviko is designed as a **two-sided platform**:

* 👤 **Customers** can browse services, book appointments, make payments and communicate with providers.
* 🧑‍🔧 **Providers** can manage jobs, accept requests, track earnings and interact with customers.

The app uses a **role-based architecture**, allowing seamless switching between customer and provider modes within the same application.
